#  EDA on Mental Health & Social Media Dataset

## Overview
This project explores how social media usage‚Äîespecially **daily screen time**, **content interaction**, and **sleep patterns**‚Äîimpacts **mental health**.  
Through Exploratory Data Analysis (EDA), we study relationships between **stress**, **happiness**, and **sleep quality** to understand behavioral patterns.

---

##  Dataset Description
The dataset includes:

- **Daily_Screen_Time (hrs)**
- **Stress_Level**
- **Happiness_Score**
- **Sleep_Quality**
- **Content_Type (Positive / Negative / Neutral)**
- **Weekday / Weekend Usage**

---

##  Objectives
- Analyze screen time behavior  
- Explore trends affecting stress and happiness  
- Understand the role of sleep on mental well-being  
- Visualize relationships using statistical plots

---

## Key Insights
- Higher **screen time** is linked to **higher stress** and **lower sleep quality**.  
- **Stress** and **happiness** show a clear **inverse relationship**.  
- Better **sleep quality** strongly correlates with higher **happiness scores**.  
- Users engaging with **positive content** show better mental health indicators.
